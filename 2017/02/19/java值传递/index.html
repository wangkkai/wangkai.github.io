java值传递

对于java来说，只有值传递一种方式，并不存在引用传递，无论是传递的参数是基本数据类型还是对象类型。对于基本数据类型传递的是其值的副本，对于对象类型传递的是引用的副本。

下面我就来举例说明。

1.基本数据类型

之所以输出0，是因为fermin(int i)传入的参数i并不是地址，而是i的一个副本，所以在函数内无论进行什么样的计算都只是改变这个副本的值，而并不会改变实际的i值。i=i++就是先赋值在自增的原因。
